# FAT12

`fat12/` 是一段用于学习文件系统分配过程的 FAT12 风格模拟代码。
它在内存中维护固定大小的“磁盘”、FAT 表、空闲块链表和简化文件目录表，通过命令行输入演示分配、查询和释放流程。

功能范围：

- 通过交互命令模拟文件创建、查找、删除
- 使用单链表维护空闲簇列表
- 在每次变更后打印文件目录、空闲块链表和 FAT 状态
- 不模拟文件内容、目录层级或磁盘持久化

文件说明：

- `main.c`：FAT12 模拟主流程与命令解释
- `list.h`：空闲块链表声明
- `list.c`：空闲块链表实现
- `rand.cpp`：随机指令生成器，用于快速冒烟测试
- `test.sh`：本地构建并执行随机测试的脚本
- `makefile`：C 源码的本地构建入口

支持的命令：

- `CRE <name> <length>`：创建文件并分配指定数量的簇
- `FIN <name>`：查询文件占用的簇链
- `DEL <name>`：删除文件并回收占用簇
- `exit`：退出程序

构建方式：

```bash
cd fat12
make
```

交互运行：

```bash
cd fat12
./fat12
```

快速验证：

```bash
cd fat12
sh test.sh
```

脚本会重新编译模拟器、编译随机指令生成器，并将一次样例运行输出写入 `out`。
